# C++ 學習清單 (CPP Learning Checklist)

這是一份 C++ 核心知識點的清單，用於追蹤學習進度與深度理解。

## 1. 基礎開發 (Basics)
- [ ] Basic Syntax (基本語法)
- [ ] Basic Types (基本類型)
- [ ] Variables (變數)
- [ ] Functions (函式)
- [ ] Headers & Source Files (標頭檔與源文件)
- [ ] Enum and Enum Class
- [ ] Static Local Variables (靜態區域變數)

## 2. 物件導向與抽象 (OOP & Abstraction)
- [ ] Classes (類別)
- [ ] Inheritance & Polymorphism and how it's achieved (繼承與多型及其實現方式)
- [ ] (Pure) Virtual Member Functions (純虛擬成員函式)
- [ ] Function Templates Basics (函式模板基礎)
- [ ] Class Templates Basics (類別模板基礎)

## 3. 語義與編譯器提示 (Semantics & Modern C++)
- [ ] Value Semantics, Pointer Semantics & Reference Semantics (值語義、指標語義與引用語義)
- [ ] `const`
- [ ] `constexpr`
- [ ] `auto`
- [ ] Lambdas
- [ ] `std::function`
- [ ] `std::optional`
- [ ] Designated Initializers (指定初始化器)
- [ ] Default Initialization (T t{}) (預設初始化)

## 4. 記憶體管理與安全 (Memory Management)
- [ ] Raw Pointers, `new` & `delete` (原始指標、new 與 delete)
- [ ] `std::unique_ptr`, `std::shared_ptr` (智能指標)
- [ ] Undefined Behavior (未定義行為)
- [ ] Accessing Uninitialized Memory (訪問未初始化的記憶體)
- [ ] Accessing Not-owned Memory (訪問不屬於自己的記憶體)

## 5. 標準庫與進階主題 (Standard Library & Advanced)
- [ ] `std::vector`, `std::array` (序列容器)
- [ ] `std::mutex` (多執行緒同步)
- [ ] Static Library, Dynamic Library (靜態庫、動態庫)
